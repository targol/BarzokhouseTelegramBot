# ربات رزرو اقامتگاه بومگردی «خانه برزک»

## ساختار پروژه

```
telegram-booking-bot/
├── bot.py               ← منطق اصلی ربات (نیازی به تغییر نیست)
├── config.py             ← تمام متن‌ها، اتاق‌ها، قیمت‌ها و اطلاعات تماس (اینجا رو ویرایش کن)
├── requirements.txt       ← کتابخانه مورد نیاز
├── rooms_photos/          ← عکس اتاق‌ها رو اینجا بذار
└── README.md
```

## مراحل راه‌اندازی

### ۱. ساخت ربات در تلگرام
۱. به [@BotFather](https://t.me/BotFather) پیام بده.
۲. دستور `/newbot` رو بزن و اسم و یوزرنیم دلخواه رو وارد کن.
۳. توکنی که بهت میده رو کپی کن.

### ۲. گرفتن آیدی عددی برای دریافت درخواست‌های رزرو
۱. به [@userinfobot](https://t.me/userinfobot) پیام بده تا آیدی عددی خودت رو بگیری.
۲. اگر می‌خوای درخواست‌ها به یک گروه ارسال بشه، ربات رو به اون گروه اضافه کن و آیدی عددی گروه رو (که با علامت منفی شروع میشه) بگیر.

### ۳. تکمیل فایل config.py
فایل `config.py` رو باز کن و موارد زیر رو با اطلاعات واقعی جایگزین کن:
- `BOT_TOKEN` → توکنی که از BotFather گرفتی
- `ADMIN_CHAT_ID` → آیدی عددی که در مرحله قبل گرفتی
- `WELCOME_MESSAGE`, `ADDRESS_TEXT`, `CONTACT_TEXT`, `SOCIAL_TEXT` → متن‌های دلخواه
- `LOCATION_LATITUDE` و `LOCATION_LONGITUDE` → مختصات دقیق اقامتگاه (می‌تونی از گوگل مپ بگیری: راست‌کلیک روی نقطه → کپی مختصات)
- `ROOMS` → اطلاعات هر اتاق (نام، توضیحات، قیمت، مسیر عکس)

### ۴. اضافه کردن عکس اتاق‌ها
عکس هر اتاق رو با همون اسمی که در `config.py` مشخص کردی (مثلاً `shatoot.jpg`) داخل پوشه `rooms_photos/` بذار.

### ۵. نصب و اجرا
```bash
pip install -r requirements.txt
python bot.py
```

اگر همه چیز درست باشه، در ترمینال پیام «ربات در حال اجراست...» رو می‌بینی. حالا می‌تونی در تلگرام دستور `/start` رو به ربات بزنی.

## نحوه کارکرد ربات

1. کاربر `/start` می‌زنه → پیام خوش‌آمدگویی + منوی اصلی (آدرس، تماس با ما، شبکه‌های اجتماعی، اتاق‌ها) نمایش داده میشه.
2. با زدن «اتاق‌ها» لیست ۵ اتاق (شاتوت، قالی، آبی، سرا، بالاخونه) نشون داده میشه.
3. با انتخاب هر اتاق، عکس + توضیحات + قیمت اون اتاق به همراه دکمه «درخواست رزرو» نمایش داده میشه.
4. با زدن «درخواست رزرو»، ربات از کاربر نام و سپس شماره تماس رو می‌پرسه.
5. بعد از تکمیل اطلاعات، یک پیام شامل نام اتاق، قیمت، نام کاربر، شماره تماس و آیدی تلگرامش به `ADMIN_CHAT_ID` ارسال میشه و به کاربر هم پیام تایید نشون داده میشه.

## نکات مهم برای استفاده واقعی (Production)

- این کد از روش **polling** استفاده می‌کنه که برای تست و استفاده‌های کوچک مناسبه. برای اجرای دائمی روی سرور، بهتره از یک سرویس مثل **systemd**، **Docker**، یا **PM2** برای بالا نگه‌داشتن پروسه استفاده کنی، یا با `run_webhook` به webhook سوییچ کنی.
- در حال حاضر ربات فقط درخواست رزرو رو ثبت و ارسال می‌کنه و رزرو رو به‌صورت خودکار نهایی نمی‌کنه (نیاز به تایید دستی مدیر داره). اگر بخوای تقویم رزرو، جلوگیری از رزرو دوبل، یا پرداخت آنلاین هم اضافه بشه، باید یک پایگاه داده (مثل SQLite) و منطق بیشتری اضافه کرد که خوشحال میشم در ادامه کمک کنم.
- شماره‌های تماس کاربران اطلاعات حساس هستن؛ اگر قراره این ربات رو عمومی منتشر کنی، به قوانین حریم خصوصی توجه کن.
